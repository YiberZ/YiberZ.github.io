Markdown 基本语法

· 2021-02-01 · # Markdown

Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档。
Markdown 语言在 2004 由约翰·格鲁伯（英语：John Gruber）创建。
Markdown 编写的文档可以导出 HTML、Word、图像、PDF、Epub 等多种格式的文档。
Markdown 编写的文档后缀为 .md, .markdown。

一、标题

注意 “＃” 后需要跟空格

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

二、字体

注意 “ * ” 后不要跟空格

*斜体*
**粗体**
***粗斜体***
~~删除线~~

斜体
粗体
粗斜体
删除线

三、引用

> 一级引用内容
>> 二级引用内容
>>> 三级引用内容

以此类推，多级嵌套

一级引用内容

二级引用内容

三级引用内容

四、分割线

---
或
***

五、图片

“ 图片 Alt ” 指的是当图片加载不出来时显示的文字信息

![图片 Alt](本地或网络图片链接)

六、超链接

[链接文字](链接地址)

七、列表

有序列表
1. 内容
2. 内容
3. 内容
...

无序列表
* 内容
* 内容
* 内容
...

嵌套列表
1. 内容
    * 内容
    * 内容
2. 内容
    * 内容
    * 内容

其他形式的嵌套以此类推

有序列表

  1. 内容
  2. 内容
  3. 内容
    ...

无序列表

  • 内容
  • 内容
  • 内容
    ...

嵌套列表

  1. 内容
    • 内容
    • 内容
  2. 内容
    • 内容
    • 内容

八、表格

|左对齐|居中对齐|右对齐|
|:---|:---:|---:|
|内容|内容|内容|
左对齐 居中对齐 右对齐
内容 内容 内容
内容 内容 内容
内容 内容 内容

九、代码

此处为避免 Markdown 语法转义，将标识符用 "()" 括住，实际使用时不加括号

(```) 代码所属语言
代码内容
...
(```)

<h1>Helllo World</h1>

以上列出的仅仅是 Markdown 的基本语法，除此之外，Markdown 还支持一些 HTML 元素标签、数学公式、转义符、流程图等，可以等用到的时候再查。
